# How to Install and Run Google Chrome on Linux

Google Chrome is a fast, secure, and popular web browser.

1. Install Google Chrome
sudo apt-get install google-chrome-stable
# OR
sudo dpkg -i <filename>

2. Fix Dependencies (if needed)
sudo apt-get install -f

3. Verify Chrome Path
which google-chrome-stable

4. Edit Chrome Launcher (Optional)
sudo leafpad /usr/bin/google-chrome-stable

# Add at the last line: --user-data-dir --no-sandbox

5. Run Google Chrome
google-chrome
